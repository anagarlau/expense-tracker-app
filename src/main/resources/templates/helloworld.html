Routes

Auth

Register: /register

Login: /login

Logout: /logout

Balance:

GET Balance for logged in User: /api/v1/balance -> Returns current balance

Transactions:

POST transaction acc to CATEGORYTYPE: /api/v1/:transactiontype (args: expenses, incomes) -> Returns the new balance

GET transaction by TID: /api/v1/transactions/:TID

PATCH transaction by TID: /api/v1/transactions/:TID -> Returns the new balance

DELETE transaction by TID : /api/v1/transactions/1 -> Returns the new balance

GET all transactions by category type(expense, income, all) between 2 dates: /api/v1/transactions?income=BOOLEAN&expense=BOOLEAN&from=DATESTRING&to=DATESTRING

GET all transactions in a category by CID between two dates: /api/v1/categories/transactions/:CID?from=DATESTRING&to=DATESTRING

Categories

GET list of existing categories and their Ids: /api/v1/categories/:type (expenses, incomes, all)

POST category: /categories

DELETE category & all associated transactions: /api/v1/categories/:CID -> Returns the new balance
